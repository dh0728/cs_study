# 큐 (Queue)

## 큐란?
먼저 집어넣은 데이터가 가장 먼저 나오는 성질인 **FIFO (First In First Out, 선입선출)** 구조를 지닌 자료 구조

### 시간 복잡도
* **삽입 / 삭제:** $O(1)$
* **탐색:** $O(n)$

### 큐의 주요 사용 사례
1. **CPU 스케줄링:** 작업을 기다리는 프로세스 및 스레드 행렬 관리
2. **네트워크 대기열:** 네트워크 접속을 기다리는 패킷이나 요청 행렬
3. **BFS (너비 우선 탐색) 알고리즘** 구현
4. **캐시 (Cache)** 구현

### 장단점 요약
| 장점 | 단점 |
| :--- | :--- |
| 데이터의 끝에서의 접근, 삽입, 삭제 속도가 매우 빠름. | 스택과 마찬가지로 중간에 위치한 데이터에 바로 접근하거나 탐색하는 것이 불가능. |
