# 힙 (Heap)

## 힙이란?

**완전 이진 트리** 기반의 자료구조로, 부모-자식 간에 일정한 대소 관계를 유지한다.

- **최대 힙(Max Heap)**: 부모 ≥ 자식. 루트가 항상 최댓값
- **최소 힙(Min Heap)**: 부모 ≤ 자식. 루트가 항상 최솟값

```
최대 힙          최소 힙
     9                1
    / \              / \
   7   5            3   5
  / \ /            / \ /
 4  6 3           7  4 6
```

> BST와 달리 힙은 형제 노드 간의 대소 관계를 보장하지 않는다.

---

## 완전 이진 트리

힙은 완전 이진 트리 구조를 유지한다. 마지막 레벨을 제외한 모든 레벨이 가득 차 있고, 마지막 레벨은 왼쪽부터 채워진다.

```
완전 이진 트리 (O)    완전 이진 트리 아님 (X)
      1                      1
     / \                    / \
    2   3                  2   3
   / \                          \
  4   5                          5
```

---

## 배열로 표현하는 힙

힙은 배열로 표현할 수 있다. 인덱스 1부터 시작할 때 부모-자식 관계는 다음과 같다.

```
인덱스:  1   2   3   4   5   6
값:    [ 9 | 7 | 5 | 4 | 6 | 3 ]

부모 인덱스: i // 2
왼쪽 자식:  i * 2
오른쪽 자식: i * 2 + 1
```

## 주요 연산

### 삽입

새 노드를 마지막에 추가한 뒤, 부모와 비교하며 위로 올린다 **(heapify-up)**.

```
# 최대 힙에 8 삽입
     9                  9                  9
    / \                / \                / \
   7   5    →        7   8    →          7   8
  / \ / \           / \ / \             / \ / \
 4  6 3  8         4  6 3  5           4  6 3  5
               (8 > 5이므로 swap)
```

### 삭제

루트(최댓값 또는 최솟값)를 꺼낸 뒤, 마지막 노드를 루트로 올리고 아래로 내린다 **(heapify-down)**.

```
# 최대 힙에서 루트(9) 삭제
     9                  3                  7
    / \                / \                / \
   7   5    →        7   5    →          3   5
  / \ /             / \                 / \
 4  6 3            4  6               4   6
(마지막 노드 3을        (3 < 7이므로 swap)
 루트로 이동)
```

## 파이썬에서의 힙

파이썬의 `heapq` 모듈은 **최소 힙**을 제공한다.

```python
import heapq

heap = []
heapq.heappush(heap, 5)   # 삽입
heapq.heappush(heap, 2)
heapq.heappush(heap, 8)

heapq.heappop(heap)        # 최솟값 꺼내기 → 2
```

최대 힙이 필요하면 값에 음수를 붙여서 사용한다.

```python
heapq.heappush(heap, -5)  # -5 삽입 (실제 값 5)
-heapq.heappop(heap)      # 최댓값 꺼내기 → 5
```

---

## 시간복잡도

| 연산 | 시간복잡도 |
|------|-----------|
| 삽입 | O(log n) |
| 삭제 (루트) | O(log n) |
| 최솟값/최댓값 조회 | O(1) |

---

## 정리

- 힙은 완전 이진 트리 기반의 자료구조로 최대/최소 힙으로 나뉜다
- 루트에서 항상 최솟값 또는 최댓값을 O(1)에 조회할 수 있다
- 삽입/삭제는 O(log n)이다
- 파이썬 `heapq`는 최소 힙이며, 최대 힙은 음수 변환으로 구현한다
