# 우선순위 큐 (Priority Queue)

## 우선순위 큐란?

일반 큐는 먼저 들어온 데이터가 먼저 나가는 FIFO 구조다. 우선순위 큐는 **우선순위가 높은 데이터가 먼저 나가는** 자료구조다.

```
일반 큐:       1 → 2 → 3 → 4    (입력 순서대로 출력)
우선순위 큐:   4 → 3 → 2 → 1    (우선순위 높은 순으로 출력)
```

---

## 힙과의 관계

우선순위 큐는 **추상적인 개념(ADT)** 이고, 힙은 이를 구현하는 **자료구조**다.

```
우선순위 큐 (개념)
        ↓ 구현
     힙 (자료구조)
```

힙 외에도 배열이나 연결리스트로 구현할 수 있지만, 힙이 가장 효율적이다.

| 구현 방식 | 삽입 | 삭제 | 최솟값 조회 |
|-----------|------|------|-------------|
| 정렬된 배열 | O(n) | O(1) | O(1) |
| 연결리스트 | O(n) | O(1) | O(1) |
| 힙 | O(log n) | O(log n) | O(1) |

---

## 파이썬에서의 우선순위 큐

파이썬에서는 `heapq` 또는 `queue.PriorityQueue`를 사용한다.

```python
import heapq

pq = []
heapq.heappush(pq, (3, 'C'))   # (우선순위, 데이터)
heapq.heappush(pq, (1, 'A'))
heapq.heappush(pq, (2, 'B'))

heapq.heappop(pq)   # (1, 'A') → 우선순위가 낮은 숫자가 먼저 나옴
heapq.heappop(pq)   # (2, 'B')
heapq.heappop(pq)   # (3, 'C')
```

튜플의 첫 번째 값을 우선순위로 사용한다. 숫자가 작을수록 우선순위가 높다.

---

## 활용 사례

- **다익스트라 알고리즘**: 현재까지의 최단 거리가 가장 짧은 노드를 우선 처리
- **작업 스케줄링**: 우선순위가 높은 작업을 먼저 처리
- **허프만 코딩**: 빈도가 낮은 문자를 우선 처리해 압축

---

## 정리

- 우선순위 큐는 우선순위가 높은 데이터가 먼저 나오는 자료구조다
- 힙으로 구현하는 것이 가장 효율적이다 (삽입/삭제 O(log n))
- 파이썬에서는 `heapq`로 구현하며, 튜플의 첫 번째 값을 우선순위로 사용한다
