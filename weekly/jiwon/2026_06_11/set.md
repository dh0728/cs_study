# 셋 (Set)

## 셋이란?

**중복 없는 데이터의 집합**을 저장하는 자료구조다. 수학의 집합 개념과 동일하다.

```
[1, 2, 2, 3, 3, 3]  →  {1, 2, 3}   (중복 제거)
```

---

## 파이썬에서의 셋

```python
s = set()

s.add(1)       # 삽입
s.add(2)
s.add(2)       # 중복 → 무시됨

print(s)       # {1, 2}

s.remove(1)    # 삭제

2 in s         # 존재 여부 확인 → True
```

리터럴로 생성할 수도 있다.

```python
s = {1, 2, 3}
```

---

## 주요 연산과 시간복잡도

| 연산 | 시간복잡도 |
|------|-----------|
| 삽입 | O(1) |
| 삭제 | O(1) |
| 존재 여부 확인 | O(1) |

맵과 마찬가지로 내부적으로 해시 테이블로 구현되어 있다.

---

## 집합 연산

셋은 수학의 집합 연산을 지원한다.

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

a | b    # 합집합  → {1, 2, 3, 4, 5, 6}
a & b    # 교집합  → {3, 4}
a - b    # 차집합  → {1, 2}
a ^ b    # 대칭차  → {1, 2, 5, 6}
```

---

## 맵과의 비교

| | 셋 | 맵 |
|--|----|----|
| 저장 형태 | 값만 | 키-값 쌍 |
| 중복 | 불가 | 키 중복 불가 |
| 순서 | 미보장 | 삽입 순서 유지 (3.7+) |

---

## 활용 사례

- 중복 제거: 리스트에서 고유한 값만 추출
- 멤버십 테스트: 특정 값이 존재하는지 O(1)에 확인
- 집합 연산: 두 데이터 집합의 교집합/합집합 계산

---

## 정리

- 셋은 중복 없는 데이터를 저장하는 자료구조다
- 삽입/삭제/탐색 모두 O(1)이다
- 합집합, 교집합, 차집합 등 집합 연산을 지원한다
- 순서를 보장하지 않는다
