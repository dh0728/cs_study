# 맵 (Map)

## 맵이란?

**키(Key)와 값(Value)의 쌍**으로 데이터를 저장하는 자료구조다. 키를 통해 값에 접근한다. 딕셔너리(Dictionary)라고도 부른다.

```
키       값
"name" → "Alice"
"age"  → 25
"city" → "Seoul"
```

키는 중복될 수 없고, 각 키에는 하나의 값만 대응된다.

---

## 파이썬에서의 맵

파이썬의 `dict`가 맵에 해당한다.

```python
m = {}

m["name"] = "Alice"    # 삽입
m["age"]  = 25

print(m["name"])       # 조회 → "Alice"

del m["age"]           # 삭제

"name" in m            # 키 존재 여부 → True
```

---

## 주요 연산과 시간복잡도

| 연산 | 시간복잡도 |
|------|-----------|
| 삽입 | O(1) |
| 삭제 | O(1) |
| 조회 | O(1) |
| 키 존재 여부 확인 | O(1) |

파이썬의 `dict`는 내부적으로 해시 테이블로 구현되어 있어 대부분의 연산이 O(1)이다.

---

## 순서 보장

파이썬 3.7 이상부터 `dict`는 **삽입 순서를 유지**한다.

```python
m = {}
m["c"] = 3
m["a"] = 1
m["b"] = 2

for k, v in m.items():
    print(k, v)
# c 3
# a 1
# b 2   ← 삽입 순서 유지
```

## 활용 사례

- 빈도 카운팅: 단어나 문자의 등장 횟수 집계
- 캐싱: 계산 결과를 키-값으로 저장해 재사용
- 그래프 인접 리스트 표현

---

## 정리

- 맵은 키-값 쌍으로 데이터를 저장하는 자료구조다
- 키는 중복될 수 없다
- 파이썬의 `dict`가 맵에 해당하며 삽입/삭제/조회 모두 O(1)이다
- 내부적으로 해시 테이블로 구현되어 있다
