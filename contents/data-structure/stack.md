# 스택 (Stack)

## 스택이란?

**후입선출(LIFO, Last In First Out)** 방식의 선형 자료구조다.

마지막에 들어온 데이터가 가장 먼저 나간다. 접시를 쌓아두고 위에서부터 꺼내는 구조와 동일하다.

```
push →  [ 3 ]  ← top
        [ 2 ]
        [ 1 ]
        -----
pop  →  [ 3 ]  가장 마지막에 들어온 3이 먼저 나감
```

---

## 주요 연산

- **push**: 스택 맨 위에 데이터 추가
- **pop**: 스택 맨 위의 데이터 제거 및 반환
- **peek**: 스택 맨 위의 데이터 확인 (제거하지 않음)

모든 연산은 top에서만 이루어지므로 O(1)이다.

---

## 파이썬에서의 스택

파이썬 리스트로 스택을 구현할 수 있다.

```python
stack = []

stack.append(1)   # push [1]
stack.append(2)   # push [1, 2]
stack.append(3)   # push [1, 2, 3]

stack.pop()       # pop → 3 반환, [1, 2]
print(stack[-1])  # peek → 2
```

---

## 활용 사례

- **함수 호출 스택**: 함수가 호출될 때마다 스택에 쌓이고, 종료되면 제거된다
- **뒤로 가기**: 브라우저 방문 기록, 텍스트 편집기의 실행 취소(undo)
- **괄호 검사**: 여는 괄호를 push, 닫는 괄호가 나오면 pop해서 짝을 확인

---

## 정리

- 스택은 LIFO 구조로 마지막에 들어온 데이터가 먼저 나간다
- push, pop, peek 모두 O(1)이다
- 파이썬 리스트의 append, pop으로 구현할 수 있다
